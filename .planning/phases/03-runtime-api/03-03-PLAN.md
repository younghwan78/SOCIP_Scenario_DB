**작성할 테스트 함수 목록 (함수명 고정):**

1. `test_graph_returns_200(api_client: TestClient)`:
   - GET `{BASE}/graph` → 200
   - resp.json()에 "scenario_id", "variant_id", "pipeline", "ip_catalog", "issues", "waivers", "reviews" 키 존재 확인
   - resp.json()["scenario_id"] == SCENARIO_ID
   - resp.json()["variant_id"] == VARIANT_ID

2. `test_graph_404(api_client: TestClient)`:
   - GET `/api/v1/scenarios/no-such-id/variants/no-such-vid/graph` → 404

3. `test_resolve_returns_200(api_client: TestClient)`:
   - GET `{BASE}/resolve` → 200
   - resp.json()에 "ip_resolutions", "sw_resolutions", "unresolved_requirements", "warnings" 키 존재 확인

4. `test_resolve_404(api_client: TestClient)`:
   - GET `/api/v1/scenarios/no-such-id/variants/no-such-vid/resolve` → 404

5. `test_gate_returns_200(api_client: TestClient)`:
   - GET `{BASE}/gate` → 200
   - resp.json()에 "status", "matched_rules", "matched_issues", "applicable_waivers", "missing_waivers" 키 존재 확인
   - resp.json()["status"] in {"PASS", "WARN", "BLOCK", "WAIVER_REQUIRED"}

6. `test_gate_404(api_client: TestClient)`:
   - GET `/api/v1/scenarios/no-such-id/variants/no-such-vid/gate` → 404

7. `test_view_architecture_mode(api_client: TestClient)`:
   - GET `/api/v1/scenarios/{SCENARIO_ID}/variants/{VARIANT_ID}/view?level=0&mode=architecture` → 200
   - resp.json()["mode"] == "architecture"
   - resp.json()["scenario_id"] == SCENARIO_ID
   - resp.json()["variant_id"] == VARIANT_ID
   - "nodes" in resp.json()

8. `test_view_topology_mode_returns_501(api_client: TestClient)`:
   - GET `/api/v1/scenarios/{SCENARIO_ID}/variants/{VARIANT_ID}/view?level=0&mode=topology` → 501
